# 🏡 Melkino – Luxury Real Estate Platform

A modern **Luxury Real Estate** web application built with **React**, **Vite**, and **Tailwind CSS v4**.

Melkino is a portfolio project focused on creating a premium user experience through clean UI, responsive layouts, reusable components, and modern frontend architecture. The project demonstrates how a scalable real estate platform can be built using today's frontend technologies.

---

## ✨ Features

- 🏠 Modern luxury real estate landing page
- 📱 Fully responsive design
- ⚡ Fast performance powered by Vite
- 🎨 Built with Tailwind CSS v4
- 🧩 Reusable React components
- 🔐 Login & Register pages
- 📊 User Dashboard
- 🏘️ Property listing & property details pages
- 🔄 Dynamic data integration
- 📦 Clean and maintainable folder structure

## 📌 About the Backend

The backend in this project is intentionally **minimal** and was created only to provide dynamic data and simulate real API interactions.

Its purpose is **not** to showcase backend architecture or business logic. The primary focus of this project is frontend development, including UI design, component architecture, responsive layouts, state management, and overall user experience.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/sadra-nosrati/melkino-real-estate.git
```

Navigate to the project

```bash
cd melkino-real-estate
```

Install dependencies

```bash
npm install
```

Run the frontend

```bash
npm run dev
```

Run the backend

```bash
cd server
npm install
npm run dev
```
